# AI-Driven Forex Signal Classifier Engine

## Project Overview
An end-to-end Machine Learning solution designed to classify and predict market direction using technical indicator features. The architecture extracts key market characteristics to output clean trading signals, which are served through a localized web application interface.

## Installation Steps
To set up the environment and run this project locally, execute the following commands in your terminal:

```bash
# Navigate to the project directory
cd ai_driven_forex_signal_classifier_engine

# Install the required technology application stack
pip install streamlit pandas numpy scikit-learn joblib
```

## How to Run the UI
To spin up the web interface and interact with the deployment application locally, run:

```bash
streamlit run app.py
```
