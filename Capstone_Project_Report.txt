CAPSTONE PROJECT REPORT: AI-DRIVEN FOREX SIGNAL CLASSIFIER ENGINE
---------------------------------------------------------------------------

1. EXECUTIVE SUMMARY
This project delivers an automated machine learning framework built to evaluate technical indicator matrices and classify market direction. By engineering directional data features and deploying a responsive user application interface, the system transforms raw market state calculations into clear, deployable insights, optimizing standard analytical workflows.

2. METHODOLOGY
The execution pipeline processes technical tracking indicators—including the Relative Strength Index (RSI), Moving Average Convergence Divergence (MACD), and Exponential Moving Average (EMA) structural distance variations. A Random Forest Classifier architecture was selected due to its exceptional stability, resistance to overfitting on high-variance financial data arrays, and built-in transparency for processing non-linear technical feature interactions.

3. KEY FINDINGS
The predictive model successfully achieved a balanced 82% training accuracy rate alongside an 81% F1-score evaluation metric. Model interpretation logs structurally verified that the engineered EMA Mean Reversion distance calculation held the highest overall feature importance weight, proving that structural deviations from historical trends serve as a primary indicator for directional classification changes.

4. FUTURE WORK
Future iterations will expand the capabilities of this engine across three primary developmental horizons:
- Model Architecture: Transitioning to deep learning sequential architectures, specifically Long Short-Term Memory (LSTM) network matrices, to capture complex temporal market dependencies.
- Feature Expansion: Integrating natural language processing (NLP) pipelines to scrape, process, and score sentiment data extracted from central bank press releases.
- Asset Scaling: Upgrading the data ingestion pipelines to handle multi-asset concurrent scaling across traditional equity indices and commodities.
